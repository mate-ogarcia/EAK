ng new nom_de_l_exo --skip-tests --inline-style --inline-template --standalone --skip-git

--skip-tests :
Ne génère pas les fichiers de test (.spec.ts).

--inline-style :
Place le CSS directement dans le décorateur du composant (styles: [...]).

--inline-template :
Place le HTML directement dans le décorateur du composant (template: `...`).

--standalone :
Crée le projet avec des composants standalone (sans NgModule, structure moderne).

--skip-git :
N’initialise pas de dépôt Git dans le projet.
